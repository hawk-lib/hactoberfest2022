# Count unique values

## Instructions

Given a sorted list of integers implement a function which counts the unique values.

[challenge](challenge.kt) | [solution](solution.kt)

## Examples

```kotlin
countUniqueValues([]) // 0

countUniqueValues([1, 1, 6]) // 2

countUniqueValues([-2, -1, -1, 0, 1]) // 4
```
