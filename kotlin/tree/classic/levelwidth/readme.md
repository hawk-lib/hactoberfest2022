## Instructions

Given the root node of a tree, return an list where each element is the width of the tree at each level.

[challenge](challenge.kt) | [solution](solution.kt)

## Examples

```kotlin
    A
  / |  \
B   C   D
|       |
E       F

Answer: [1, 3, 2]
```
